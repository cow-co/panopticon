# Panopticon
## An app to allow you to visualise and track your digital footprint

## What is PANOPTICON?

A digital footprint tracker which is:

- Standalone
- Privacy-respecting (I record no telemetry and I do not record any of the personal data you may enter)
- Non-internet-connecting (tested on a disconnected environment)
- Graph-based (with support for basic graph traversal operations)
- Easy-to-use
